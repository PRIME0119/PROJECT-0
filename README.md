## 📖 What is Dominus?

**Dominus** is a Raspberry Pi–based hardware-software control system designed to remotely manage and operate multiple computing environments with minimal power consumption and high reliability.

It enables on-demand activation of dual systems, hardware-level switching between multiple operating systems via relay-controlled drives, and secure remote access through redundant network paths.

The system eliminates the need for always-on high-power machines by maintaining a **low-power control node** while keeping primary systems in an off or standby state — and integrates both **web-based control** and **physical fail-safe mechanisms** to ensure continuous accessibility even in failure scenarios.

## 🧠 The Philosophy

<div align="center">

| Principle | Meaning |
|:---:|:---:|
| 🔋 **Efficiency** | High-power systems stay OFF until needed |
| 🔁 **Redundancy** | Every access path has a fallback |
| 🔩 **Hardware First** | Physical override survives any software crash |
| 🌍 **Always Accessible** | Remote control from anywhere in the world |
| 🧱 **Zero Virtualization** | Real OS on real drives — full performance |
